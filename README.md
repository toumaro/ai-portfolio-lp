# とうまろ | AI活用ポートフォリオLP

自分のプロフィール、AI活用テーマ、作ったもの、これから作りたいものをまとめた1枚もののランディングページ（LP）です。

## 概要

- **自己紹介** — プロフィールとAI活用の取り組み
- **これから作りたいもの** — 今後のテーマ
- **作ったもの** — 制作物の紹介
- **つながる** — 各種リンク

## 技術構成

- 単一の `index.html`（依存なし）
- [Tailwind CSS](https://tailwindcss.com/)（CDN）
- Google Fonts（Zen Kaku Gothic New / Murecho / JetBrains Mono）
- ダークテーマ・レスポンシブ対応

ビルドや開発サーバーは不要です。

## 見る

`index.html` をブラウザで直接開いてください。

```
open index.html
```

## ディレクトリ構成

```
.
├── index.html      # LP本体
├── assets/         # 画像・ロゴなど
├── CLAUDE.md       # 制作ルール
└── README.md
```

## ライセンス

個人のポートフォリオ用途です。
